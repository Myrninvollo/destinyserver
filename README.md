Pokemon Destiny Server
=============

HOWTO COMPILE:  
Install Eclipse IDE.  
Export > Jar File.  
Make sure ONLY the src folder is checked at resources to export.  
Use existing manifest > MANIFEST.MF  
  
  
See Wiki for detailed instructions.
